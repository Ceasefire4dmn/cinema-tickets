# Cinema Tickets
**Веб-приложение для выбора сеансов и покупки билетов в кино. Проект выполнен на React с использованием современных инструментов фронтенд-разработки.**

**Стек технологий:**
- Vite - быстрая сборка проекта.
- React.
- Tailwind CSS - современные стили и плавная анимация интерфейса.
- Lucide React - набор минималистичных иконок.
- Context API - централизованное управление состоянием данных (DatabaseProvider).

Текущий статус:
На данный момент реализован функционал просмотра афиши, выбора фильмов и работы с залами, покупки билетов с выбранным временем сеанса и места (после оплаты места становятся недоступными для повторного выбора).

⚠️ Важно: Страницы «Оплата» и «Билет» (контроль произведенных оплат и купленных билетов) в данный момент находятся в активной стадии разработки.
