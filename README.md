# Cinema Tickets
**Веб-приложение для выбора сеансов и покупки билетов в кино. Проект выполнен на React с использованием современных инструментов фронтенд-разработки.**

[Ссылка на деплой Кинотеатра](https://ceasefire4dmn.github.io/cinema-tickets/)

**Стек технологий:**
- Vite - быстрая сборка проекта.
- React.
- Tailwind CSS - современные стили и плавная анимация интерфейса.
- Lucide React - набор минималистичных иконок.
- Context API - централизованное управление состоянием данных (DatabaseProvider).

Текущий статус:
На данный момент реализован функционал просмотра афиши, выбора фильмов и работы с залами, покупки билетов с выбранным временем сеанса и места (после оплаты места становятся недоступными для повторного выбора).

**Важно**: Страницы «Оплата» и «Билет» (контроль произведенных оплат и купленных билетов) в данный момент находятся в стадии активной разработки.

## Быстрый запуск
Для запуска проекта на локальном компьютере убедитесь, что у вас установлена Node.js (рекомендуемая версия 18.x и выше).

1. Клонирование репозитория
Откройте терминал и выполните команду для копирования проекта в выбранную папку:

git clone https://github.com/Ceasefire4dmn/cinema-tickets.git

Перейдите в директорию проекта:
cd cinema-tickets

2. Установка зависимостей 
Установите все необходимые библиотеки и пакеты, указанные в package.json:

npm install

3. Запуск в режиме разработки 
Запустите локальный сервер для просмотра проекта:

npm run dev

После запуска проект будет доступен по адресу: http://localhost:5173 (или другому порту, указанному в терминале).
